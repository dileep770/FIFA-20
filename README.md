# 📌 Project Overview
The FIFA 20 Player Data Analysis project explores the FIFA 20 Career Mode dataset to analyze player performance, technical skills, physical attributes, and overall ratings. The project involves data cleaning, preprocessing, exploratory data analysis (EDA), visualization, and K-Means clustering to identify patterns among players. It also answers key business questions related to player demographics, performance trends, and positional comparisons, providing data-driven insights for football analytics.

# 🎯 Problem Statement
The objective of this project is to analyze the FIFA 20 Career Mode dataset to uncover meaningful insights into player performance and characteristics. The project focuses on performing exploratory data analysis, answering business-related questions, and clustering football players based on their attributes using machine learning techniques to support data-driven football analytics.
